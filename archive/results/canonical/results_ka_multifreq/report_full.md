## 2. 引言

在前一阶段的工作中，许多实验、图像说明和后续笔记本都默认围绕 `22.5 GHz` 这一参考频点展开。这样的做法在单点验证阶段是有效的，因为它能够让传播链路、接收机恢复和导航解算的接口先稳定下来，但它也天然带着一个局限，那就是它只能解释一个频点的行为，不能回答整个泛卡频段上的规律。只要后续算法设计、参数整定和实验判断仍然建立在单点观察之上，就很容易把“这个频点的表现”误当成“整个频带的共性”，从而在推广到其他频点时出现解释偏差。

这也是本报告要把频率本身提升为主分析维度的原因。这里真正关心的，不是某个频点是否偶然表现较好，而是传播层中的振幅、相位和群时延在频率变化时如何系统性地改变，接收机恢复过程又如何把这些传播变化转化为码时延恢复误差、频率恢复误差和锁定稳定性的差异，最终这些差异再如何沿着单历元定位与多历元动态估计继续向下传递。换句话说，本报告研究的是一条跨层传递链，而不是若干彼此分离的单点数值。

因此，全频研究的意义并不只在于把已有图重新画成跨频曲线，而在于为后续所有依赖 `22.5 GHz` 的笔记本和实验提供新的基线：以后讨论传播恢复、困难场景定位和动态滤波时，频率不能再被视为一个固定常数，而必须被视为影响系统行为的一条主轴。只有这样，后续基于整个泛卡频段的实验扩展才有统一而自洽的解释框架。

## 3. 研究背景与关键概念解释

### 3.1 研究对象

本报告研究的是“真实单通道传播背景之上的多层导航实验”。这里所说的“多层”，并不是把几类图像简单拼接在一起，而是指同一套真实单通道传播背景被连续映射到了三个层面之中。最上面的一层是传播与接收层，在这一层中，电子密度场通过几何光学近似传播模型决定了信号振幅、相位和群时延的变化，接收机再把这些变化转化为可观测的恢复误差和锁定稳定性。中间一层是单历元定位层，在这一层中，单通道统计被继续映射为伪距观测精度，并在多星几何条件下形成单次求解的位置误差。最下面的一层是多历元动态估计层，这一层把伪距和距离变化率放到连续时间序列中联合利用，最终给出更接近真实导航应用场景的动态位置与速度误差。

### 3.2 关键概念

几何光学近似传播模型（WKB）是本报告最上游的传播描述工具。它的作用，是把电磁波在缓变介质中的传播近似写成振幅、相位和群时延三个量，从而描述信号穿过等离子体环境以后，传播强弱、相位积累和码观测相关时间延迟会发生什么变化。若把传播结果写成复传输系数，则可以写成

```text
h_p(t, f) = A(t, f) · exp(j φ(t, f))
ω = 2π f
τ_g(t, f) = - ∂φ(t, f) / ∂ω
```

这里的 `A(t, f)` 是传播振幅，`φ(t, f)` 是传播相位，`τ_g(t, f)` 是群时延。群时延不是额外引入的经验量，而是由相位对角频率求导得到的，因此它本质上描述的是传播相位随频率变化时，码观测会感受到的有效时延扰动。当前工程实现并不是直接对连续函数求导，而是按照相邻频点的中心差分计算群时延，因此在报告所用的数据中，实际采用的是

```text
τ_g(t, f) ≈ - [φ(t, f + Δf) - φ(t, f - Δf)] / (4πΔf)
```

这也是为什么本报告中的群时延分析能够自然接到全频频点网格之上，因为它本来就是通过频率邻点之间的相位差分获得的。

进入接收机以后，传播结果不再只是停留在 `A(t)`、`φ(t)` 和 `τ_g(t)` 这些上游量上，而是通过接收信号模型进入捕获与跟踪环路。当前链路可以概括写成

```text
r(t) = A(t) · c(t - τ_total(t)) · exp(j φ_total(t)) + n(t)
τ_total(t) = τ_geom(t) + τ_g(t)
```

其中 `c(t)` 是发射扩频信号，`τ_geom(t)` 是几何码时延，`τ_g(t)` 是色散传播导致的附加群时延，`φ_total(t)` 则把运动相关相位和传播相位合并到一起。正因为传播层是以这样的方式进入接收机，所以后相关信噪比、载波锁定指标以及码时延和频率恢复误差才会成为传播层影响导航层的第一道可见接口。后相关信噪比反映的是相关器输出端的信号与噪声之比，载波锁定指标描述的是载波跟踪环是否稳定跟住真实载波，均方根误差则用来压缩描述全时段的恢复精度。

导航层中的伪距，是接收机由码相位恢复出的距离型观测量。它并不只是几何距离，还混有时钟差、传播延迟和测量噪声，因此当单通道恢复变差时，伪距质量会直接下降。多普勒则是载波频率因为相对运动和传播环境变化而发生的偏移，在动态滤波层中主要体现为距离变化率信息。单历元加权最小二乘（WLS）的作用，是在一次求解中让质量更高的观测占更大权重、质量更差的观测占更小权重，从而抑制低仰角和低信噪比观测造成的误差放大；扩展卡尔曼滤波（EKF）则是一种逐历元递推估计方法，它把上一时刻状态预测与当前观测结合起来，同时估计位置、速度和时钟状态，因此更适合描述连续时间条件下的动态导航表现。

### 3.3 结果边界

本报告必须严格保持一个边界意识，那就是这里研究的是“真实单通道传播背景之上的跨层导航规律”，而不是“真实多星端到端系统的最终工程定标”。单通道传播与接收机恢复部分是直接复用的真实链路，因此传播层到接收层之间的关系具有直接物理基础；但多星几何和动态轨迹仍然是自洽构造的，不是广播星历和真实飞行轨迹，各卫星也没有各自独立的真实传播路径，而是共享单通道真实背景后再映射到导航层。正因为如此，本报告的价值在于揭示频率规律、说明层间传递关系，并为后续更真实的多星扩展提供依据，而不是在这里直接宣布系统级最终能力。

### 4.2 三层链路

本次实验链路从电子密度场出发，先得到真实传播振幅、相位和群时延，再进入单通道接收机的捕获与跟踪过程，形成单通道伪距、载波与多普勒观测，随后这些观测被映射到多星标准伪距构造之中，继续向下进入单历元加权定位和多历元动态滤波，最终汇总为全频综合指标和本报告所使用的图表。这样组织的意义，不是为了把流程写得更长，而是为了保证每一层都能单独观察，同时又能清楚地解释上一层的变化如何沿着同一条链路继续向下传递。

在单历元导航层，标准伪距观测并不是把单通道内部的 `c · τ_est` 直接原样抬上来，而是统一写进标准观测方程。对第 `i` 颗卫星，可写成

```text
ρ_i = ||r_{s,i} - r|| + c(δt_r - δt_{s,i}) + d_trop,i + d_disp,i + b_hw,i + ε_{ρ,i}
```

这里 `||r_{s,i} - r||` 是几何距离，`δt_r` 和 `δt_{s,i}` 分别是接收机和卫星时钟项，`d_trop,i`、`d_disp,i` 和 `b_hw,i` 分别对应对流层项、色散传播项和硬件偏差，`ε_{ρ,i}` 则吸收了单通道恢复误差在导航层的等效表现。也正因为如此，单通道恢复结果在这里并不是被当作标准伪距本体，而是以误差统计和非几何项的形式进入统一模型。

到了动态层，距离变化率观测与伪距一起进入观测方程。其主线表达可写成

```text
ρdot_i = u_LOS,i^T (v_{s,i} - v_r) + c(δtdot_r - δtdot_{s,i}) + ε_{ρdot,i}
range_rate_mps = -(c / f_c) · doppler_hz
```

其中 `u_LOS,i` 是视线方向单位向量，`v_{s,i}` 和 `v_r` 分别是卫星与接收机速度，`δtdot_r` 与 `δtdot_{s,i}` 是钟漂项。报告后面提到的距离变化率观测，正是按上述换算从单通道多普勒频移映射而来，因此它不是一个额外凭空构造的量，而是链路内部频率观测进入标准导航方程后的形式。

如果说上一张图回答的是“收益有没有传下来”，那么这一张图回答的是“这种传递主要发生在哪一段频带”。本节只保留一个主判断：系统的改善不是平均铺开的，而是先在低频到过渡段完成大幅收缩，再在高频段把动态估计稳定性继续拉开。

![频段统计对比图](report_visuals/band_summary_metrics.png)

按频段看，低频段与高频段之间已经拉开了清晰差距：单通道码时延均方根误差均值从 `764.461` 纳秒降到 `534.584` 纳秒，困难场景加权定位三维误差均值从 `159.855` 米降到 `52.785` 米，动态滤波三维位置误差均值则从 `126.632` 米压到 `2.166` 米。更关键的是，这种改善分成两个阶段完成。从 `19.0 GHz` 到 `22.5 GHz`，系统先完成一次大幅去劣化；从 `22.5 GHz` 到 `31.0 GHz`，单通道层的继续改善已经没有前一阶段那么剧烈，但动态层仍然能继续大幅受益，这说明高频端的价值不只是让接收机更容易工作，而是让已经改善过的观测条件更稳定地转化为递推估计优势。与此同时，图上仍保留了 `19.5 GHz`、`20.0 GHz`、`21.5 GHz`、`22.0 GHz`、`25.0 GHz`、`29.5 GHz` 和 `30.0 GHz` 一带的局部起伏，因此后文的所有解释都继续遵守“总体改善叠加局部波动”的框架，而不把全频结果写成理想化的单调曲线。

## 6. 分层结果分析

### 6.1 单通道传播与恢复层

在这一层，最值得关注的不是单独某一个传播量，而是“传播量的变化最终是否变成了接收机可见的恢复误差”。从定义上说，群时延满足 `τ_g = -∂φ/∂ω`，因此它刻画的是相位随频率变化的斜率，而不是直接等于接收机要看到的码时延恢复误差。现有结果表明，群时延中位数从低频端到高频端的整体变化并不算剧烈，但码时延恢复误差却在低频端显著偏大，这说明接收机恢复层对传播变化的响应并不是线性的直接映射。传播层的扰动必须先经过 `r(t) = A(t) · c(t-τ_total(t)) · exp(jφ_total(t)) + n(t)` 这一信号形成过程，再进入捕获与跟踪环，最后才被表现为码时延恢复误差、频率恢复误差和锁定稳定性差异，因此只看传播层本身的数值大小，无法完整解释导航层的误差差异。真正重要的，是传播层扰动在环路里被怎样放大、吸收或重新分配，而这一过程正是低频段和高频段表现分化的起点。

### 6.2 单历元加权定位层

这一层先给出结论：单历元定位对频率变化的敏感性明显高于传播层本身，因为上游观测误差一旦进入加权求解，就会在困难场景里被迅速放大或压缩。图中真正值得看的，不是某一条曲线单独怎么走，而是观测统计、困难场景误差和分位数是否同步向更稳定的区间移动。

![单历元定位全频趋势图](wls/cross_frequency/wls_metrics_vs_frequency.png)

从图上看，色散指标本身的变化并不夸张，但伪距统计、一秒平滑标准差、困难场景定位误差和九成分位值却都随着频率升高整体收缩，说明真正被放大的不是某个传播量，而是传播扰动经过观测构造和权重分配之后的结果。尤其在困难场景中，普通最小二乘与加权最小二乘之间的差异在低频端更明显，而在高频端整体下压，这意味着低仰角和弱观测会最先暴露上游质量差异；一旦单通道统计改善，权重矩阵就能更有效地限制差观测的影响，单历元解才会更早进入收敛区。

这一层在数学上对应的是带权线性化最小二乘问题。若把单历元观测残差写成 `Δρ`，设计矩阵写成 `H`，未知增量写成 `Δx`，则可以把求解写成

```text
Δx = arg min || W^(1/2) (Δρ - HΔx) ||_2^2
Δx = (H^T W H)^(-1) H^T W Δρ
```

这里的 `W` 不是任意指定的权重矩阵，而是由单通道观测误差统计映射而来，因此它直接承接了上游传播与接收层的质量信息。也正因为如此，当单通道伪距统计较差时，即便已经采用加权求解，也很难把误差完全压住，因为进入求解器之前观测本身的质量已经显著退化；而当单通道统计改善以后，加权求解就能更有效地抑制低质量观测带来的误差放大。因此，单历元加权定位的改善并不是单靠求解器本身完成的，它一方面依赖传播恢复层能否把观测质量抬起来，另一方面也依赖权重映射是否足够准确，能够把较差观测真正压到较低影响力。

### 6.3 多历元动态滤波层

这一层的主结论更集中：高频端真正拉开差距的地方，不在单历元求解器，而在多历元递推能否持续利用更稳定的伪距和距离变化率观测。图中需要优先看的不是绝对最小值，而是三类解是否形成了稳定分层，以及这种分层从哪个频段开始明显出现。

![动态滤波全频趋势图](ekf/cross_frequency/ekf_metrics_vs_frequency.png)

图中的四个子图共同说明，只有把距离变化率观测一并纳入以后，动态层才真正开始稳定兑现高频收益。位置误差图显示，仅伪距解在全频上都明显高于“伪距加距离变化率”解，并且低频端峰值极高；速度误差图进一步说明，没有距离变化率时速度约束几乎立不住；新息均值和仅预测历元数则表明，高频端不仅误差更小，而且观测更新本身更一致、被迫只做预测的时刻更少。因此，这一层的重点不是证明滤波器形式本身，而是说明当观测质量和距离变化率信息同时改善后，递推结构才真正进入稳定工作区。

这一层在模型上使用的是八状态常速度滤波器。状态向量写成

```text
x_k = [r_x, r_y, r_z, v_x, v_y, v_z, c_b, c_d]^T
```

其中前三维是位置，中间三维是速度，最后两维分别是钟差和钟漂。对应的状态转移关系可写成

```text
r_{k+1}   = r_k + v_k Δt
v_{k+1}   = v_k + w_v
c_{b,k+1} = c_{b,k} + c_{d,k} Δt
c_{d,k+1} = c_{d,k} + w_c
```

当前实现里，过程噪声并不是简单常数，而是按白加速度和钟漂随机游走离散化。若写成二阶积分白噪声块，则可以表示为

```text
Q_block = σ^2 · [ Δt^4/4   Δt^3/2 ]
                [ Δt^3/2    Δt^2  ]
```

预测与更新则遵循标准递推形式：

```text
x_{k|k-1} = F x_{k-1|k-1}
P_{k|k-1} = F P_{k-1|k-1} F^T + Q
K_k = P_{k|k-1} H_k^T (H_k P_{k|k-1} H_k^T + R_k)^(-1)
x_{k|k} = x_{k|k-1} + K_k (z_k - h(x_{k|k-1}))
P_{k|k} = (I - K_k H_k) P_{k|k-1}
```

这背后的逻辑并不复杂，但必须沿着层间关系来看。当单通道恢复更稳定时，距离变化率观测更容易形成有效约束，观测协方差 `R_k` 也更容易保持在较合理的量级；而动态滤波本身又能够把相邻历元的信息串联起来，因此比单历元求解更能消化局部噪声扰动。也正因为如此，当传播恢复层和距离变化率观测都处在较稳定的状态时，位置误差就会在动态滤波层被快速压缩到很低水平。高频段之所以在这一层表现尤其突出，并不是因为滤波器“自动变强”了，而是因为它终于拿到了更适合被递推利用的观测条件。

## 7. 频段分区结论

### 7.1 分区依据

本报告不采用任意切段的方式，而是固定把全频结果分为三个部分来讨论，即 19.0 GHz 至 22.0 GHz 的低频段、22.5 GHz 至 25.5 GHz 的过渡段，以及 26.0 GHz 至 31.0 GHz 的高频段。这样分区并不是为了让结构看起来整齐，而是因为现有全频结果已经显示，这三个频段在单通道恢复、单历元定位和动态滤波三层上都呈现出可区分的统计行为，既有明显的总体差异，也各自保留了不同的局部波动特征。

### 7.3 分区讨论

### 低频段

该频段覆盖 19.0 至 22.0 GHz，共 7 个频点。

这一段最突出的问题是单通道码时延恢复误差大，而且不同频点之间起伏明显。传播层的时延和衰减虽然并不呈现灾难性恶化，但接收机恢复过程对这些变化更敏感，最终把误差放大到定位层与动态滤波层。就统计量而言，这一段的单通道码时延均方根误差均值达到 `764.461` 纳秒，困难场景加权定位三维误差均值为 `159.855` 米，动态滤波三维位置误差均值为 `126.632` 米，整体处于三个频段中最不稳定的位置。若把它放回公式框架里理解，可以说低频段中 `τ_g` 虽然不是唯一主因，但它进入 `τ_total` 之后更容易在接收机环路中放大为较大的 `ε_ρ`，随后再通过较不理想的 `W` 与 `R_k` 向下扩散。即便在这一段内部，不同频点之间的差异也很大，例如动态滤波表现最好的 `21 GHz` 位置误差仍有 `56.657` 米，而表现最差的 `19.5 GHz` 则达到 `256.882` 米，这说明低频段的问题并不是单一频点失常，而是整个频段都更容易把传播层的不利变化持续放大。


### 过渡段

该频段覆盖 22.5 至 25.5 GHz，共 7 个频点。

这一段体现的是从“可用”向“稳定好用”的过渡。单通道误差继续下降，困难场景定位误差明显收缩，但仍保留若干局部起伏，说明传播层改善尚未完全转化为稳定的导航层收益。这里的单通道码时延均方根误差均值已经降到 `573.727` 纳秒，困难场景加权定位三维误差均值收缩到 `70.666` 米，动态滤波三维位置误差均值也进一步压低到 `26.519` 米，整体显著优于低频段。从模型上看，这意味着 `ε_ρ` 和 `ε_ρdot` 的量级已经明显下降，使得加权矩阵 `W` 和动态层观测协方差 `R_k` 开始稳定地发挥作用。然而，这一段仍然不能被简单描述为“已经稳定”，因为动态滤波最好时在 `25.5 GHz` 可以做到 `1.850` 米，而 `25.0 GHz` 仍然会回升到 `56.189` 米，说明过渡段的本质并不是已经完全进入稳定区，而是传播改善开始持续显现，但系统仍会受到局部几何和门控条件的明显扰动。


### 高频段

该频段覆盖 26.0 至 31.0 GHz，共 11 个频点。

这一段已经进入明显稳定区。单通道误差总体维持在较低水平，单历元定位误差显著收缩，动态滤波的三维位置误差降到数米甚至一米左右，说明距离变化率观测在这一段能被更充分地利用。从统计结果看，这一段的单通道码时延均方根误差均值为 `534.584` 纳秒，困难场景加权定位三维误差均值为 `52.785` 米，动态滤波三维位置误差均值进一步压缩到 `2.166` 米，已经和前两个频段拉开明显差距。若回到状态空间模型来看，这说明高频段更容易同时给出较稳定的 `ε_ρ` 与 `ε_ρdot`，从而让预测项 `F x_{k-1|k-1}` 与观测更新项 `K_k(z_k-h(x))` 之间形成更有效的互补。更重要的是，这一段内部虽然仍然存在局部起伏，但波动尺度已经明显收敛，例如动态滤波最好的 `30 GHz` 位置误差为 `0.855` 米，最差的 `28.5 GHz` 也只有 `4.954` 米。这样的结果说明，高频段的优势并不是某个单点偶然特别好，而是整个频段都更容易把传播层改善转化为稳定的动态定位收益。

动态层则把过渡态进一步坐实。`22.5 GHz` 下，伪距加距离变化率动态滤波平均三维位置误差已经降到 `15.091` 米，比 `19.0 GHz` 大幅改善，但离 `31.0 GHz` 的 `1.309` 米仍有明显差距。轨迹误差图中，蓝线已经能较快贴近零附近，说明递推开始稳定受益；但仅伪距解仍然漂移，新息图在中间若干时段仍有尖峰，对比图中的黑线也还带着明显脉冲。这说明 `22.5 GHz` 的价值在于它足以证明“伪距加距离变化率”方案是有效的，却还不能说明系统已经进入最稳定的工作区，因此它适合作为方法验证点，却不再适合作为唯一频点来代表全频规律。

## 9. 讨论与局限

### 9.1 为什么高频段总体更优

从现有结果看，高频段总体更优并不只是因为某一个传播量单独变好了，而是因为三个层面的因素同时朝着有利方向变化。首先，接收机恢复层的码时延误差整体收缩，使得进入导航层的伪距统计质量明显提高；其次，单历元定位层在困难场景中的误差放大幅度下降，加权求解不再需要在极差观测和不理想几何之间艰难平衡；最后，动态滤波层能够更稳定地利用距离变化率信息，使连续历元处理真正发挥出递推约束的优势。正是这三层因素同时改善，才构成了高频段稳定优于前两个频段的根本原因。

### 9.2 为什么仍会出现局部波动

局部波动并不意味着总体趋势失效，恰恰相反，它说明这个系统从传播到导航是多因素耦合的。传播层变化本身就不是严格线性单调的，进入单历元求解以后又会受到几何条件和权重映射的影响，而到了动态滤波层，还会继续受到观测门控和仅预测历元的影响。因此，本报告采用的是“总体趋势叠加局部波动”的解释框架，而不是把所有曲线强行描述为简单单调。只有保留这种波动视角，后续基于全频结果开展实验时，才不会在局部反弹出现时误判整个趋势。

### 9.3 本报告不能替代什么

本报告不能替代真实多星端到端系统定标，因为这里的多星几何不是广播星历，各卫星也没有独立的真实传播路径，动态轨迹同样不是实飞轨迹。也就是说，这份报告的目标从来不是宣布某个系统最终能达到什么工程指标，而是回答“频率变化会怎样影响跨层链路”这个问题。不过，虽然它不能替代最终定标，这份报告已经足够支撑两类关键工作：一类是把后续笔记本和实验从单频点扩展为全频分析，另一类是为更进一步接入真实星历、真实轨迹和独立传播路径提供频率侧依据。它的价值不在于终局，而在于为下一阶段提供一套已经被全频数据支持的解释基线。

## 10. 结论

本次全频研究最重要的结论，是频率不能再被视为一个固定常数，而必须被视为后续所有实验和笔记本的主实验维度。只围绕单一频点构建工作流，已经不足以解释传播层变化如何进入接收机恢复，更不足以说明这些变化在单历元定位和多历元动态估计中究竟会如何继续放大或收敛。

沿着本报告使用的跨层链路来看，单通道传播恢复误差是传播层影响导航层的关键中间环节。传播层的振幅、相位和群时延变化并不会原封不动地等于导航误差，而是先经过接收机恢复环节，被重新表现为码时延恢复误差、频率恢复误差和锁定稳定性差异，再继续下传到伪距统计质量之中。也正因为如此，困难场景单历元定位才会成为跨频比较中最敏感的一层，它最能把观测质量和几何条件共同作用后的误差放大机制显露出来。

最终的动态滤波结果则说明，高频段最有价值的地方，不只是数值更小，而是它更容易把传播层改善稳定地转化为连续历元定位收益。低频段更多体现为误差放大和波动积累，过渡段体现为明显改善但仍伴随局部起伏，而高频段则表现为误差底座的系统性下移和动态约束利用效率的显著提升。这也是为什么后续工作不应再以 `22.5 GHz` 作为唯一中心，而应当真正面向整个泛卡频段组织分析与实验。
